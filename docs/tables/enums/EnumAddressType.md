# AddressType Enum

Bit mask, combines address type (EAddressType) and owner type (EOwnerType)

* Unknown = 0
* ContactPostalAddress = 1
* ContactStreetAddress = 2
* PersonPrivateAddress = 16387
* QuoteBillingAddress = 8196
* QuoteShippingAddress = 8197
