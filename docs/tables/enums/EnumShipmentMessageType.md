# ShipmentMessageType Enum

Identifies if HTML or plain message is included in message.

* Plain = 0
* Html = 1
* SourceView = 2
* Sms = 4
* Document = 8

## Usage
* [s_message](s_message.md).flags - A message used in a shipment. Can be html and/or plain text
