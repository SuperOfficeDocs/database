# PhoneType Enum

Bit mask, combines phone type (EPhoneType) and owner type (EOwnerType)

* Unknown = 0
* ContactPhone = 1
* ContactFax = 3
* PersonDirectPhone = 16385
* PersonDirectFax = 16387
* PersonPrivate = 16388
* PersonMobile = 16389
* PersonPager = 16390
